번호 목록을 그대로 붙여넣어도 됩니다

데모 버전은 실제 링크를 만들지 않습니다. 유튜브, 블로그, 논문, 이미지 자료는 검색 키워드와 추천 플랫폼 중심으로 표시하며, 실제 API 연결 후 URL과 출처를 붙일 수 있습니다.

목표를 입력하면 분석 카드가 생성됩니다.

목표가 많으면 오늘 안에 끝낼 항목과 보류할 항목을 나눕니다. 실행 가능성이 낮은 목표는 더 작은 행동 단위로 쪼개서 제안합니다.
