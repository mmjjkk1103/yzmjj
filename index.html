오늘의 방향을 설계하는 AI

짧게 적은 목표를 실행 가능한 계획으로 바꿔드립니다. 막연한 생각을 목표 해석, 실행 방법, 공부 방향, 우선순위, 결과물로 정리합니다.

번호 목록을 그대로 붙여넣어도 됩니다

현재 버전은 실제 AI API를 호출하지 않고, 입력 문장을 규칙 기반으로 분석하는 데모입니다. 나중에 API 연결 시 requestAiAnalysis 함수만 교체하면 됩니다.

목표를 입력하면 분석 카드가 생성됩니다.

목표가 많으면 오늘 안에 끝낼 항목과 보류할 항목을 나눕니다. 실행 가능성이 낮은 목표는 더 작은 행동 단위로 쪼개서 제안합니다.
